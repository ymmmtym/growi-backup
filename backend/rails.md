# Rails
MVC

## Install

```shell=
gem install rails -v '6.0.0'
```

## Create App
```shell=
rails new ${APP}
cd ${APP}

rails s # rails server
rails c # rails console
rails g model # rails generate model
rails g controller
rails g scaffold
rails destroy
rails dbconsole
```
